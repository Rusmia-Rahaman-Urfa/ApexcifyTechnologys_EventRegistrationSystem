# 🚀 EventHub | Full-Stack Engineering Portal
EventHub is a high-performance, full-stack event registration system designed with a modern Cyber-Purple aesthetic. It allows users to browse engineering sessions, register for events, and enables admins to manage the event lifecycle through a secured dashboard.

# 🎨 Design Philosophy
The application features a Deep Royal Purple & Orchid theme, utilizing:

***Glassmorphism:*** Frosted glass panels with subtle purple borders.

***Glow Effects:*** Interactive UI elements with CSS text-shadow and box-shadow glows.

***Responsive Layout:*** Fully optimized for mobile, tablet, and desktop using Tailwind CSS.

***Micro-interactions:*** Smooth hover transitions and card lift animations.

# 🛠️ Tech Stack
## Frontend
***HTML5 & CSS3:*** Structural foundation.

***Tailwind CSS:*** Modern utility-first styling.

***JavaScript (ES6+):*** Dynamic DOM manipulation and Fetch API.

***SweetAlert2:*** Professional, themed modal dialogs.

## Backend
***Node.js & Express:*** Scalable server-side logic.

***MongoDB & Mongoose:*** NoSQL database for flexible data modeling.

***Cors & Body-Parser:*** Middleware for secure cross-origin requests.

## 🔑 Key Features
***Secure Authentication:*** Simple email-based login system.

***Real-time Event Grid:*** Fetches and displays available events dynamically from MongoDB.

***Dynamic Search:*** Instant filtering of events by title or category.

***User Registration:*** Seamless flow for users to book spots in specific events.

## Admin Control Center:
***Deploy Events:*** Create new events with titles and descriptions.

***Registry Data:*** View a live list of all user registrations.

***Erase Events:*** Remove outdated or completed events from the database.

***Personal Schedule:*** Users can view their own specific bookings
